# SAMI — UI/UX Component Strategy (2026)

## Framework Selection
- **shadcn/ui**: Core application foundation — robust, accessible components
- **Magic UI**: Polished micro-interactions and subtle animations
- **Aceternity UI**: High-impact visual effects for landing pages / hero sections

## Design Principles
- Dark theme as default with zinc/gray palette
- Monospace font for terminal-like interfaces
- Clickable buttons for all common terminal commands
- Lucide icons throughout
